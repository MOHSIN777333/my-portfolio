# Mohsin Ali — Personal Portfolio

A clean and responsive personal portfolio website built with React.js and Tailwind CSS.

🔗 **Live Demo:** [my-portfolio-ten-wheat-46.vercel.app](https://my-portfolio-ten-wheat-46.vercel.app/)

---

## ✨ Features

- Responsive design — works on all screen sizes
- Hero section with introduction
- About Me section
- Skills showcase
- Projects section
- Contact section

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/MOHSIN777333/mohsin-portfoloi.git

# Install dependencies
npm install

# Run locally
npm run dev
```
